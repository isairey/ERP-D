# 🏢 ERP Management System

<p align="center">
  <b>Sistema ERP modular para la gestión integral empresarial</b>
</p>

---

## 📌 Descripción

Este sistema ERP está diseñado bajo una arquitectura **modular**, permitiendo a las empresas implementar únicamente los componentes que necesiten según sus operaciones.

El sistema se compone de **9 subsistemas principales**, enfocados en cubrir todas las áreas clave de gestión empresarial.

---

## 🧩 Módulos del sistema

| 🧱 Subsistema     | ⚙️ Funcionalidades principales                                                                                                |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| 📦 Compras        | Gestión de inventario de materiales, precios de proveedores, historial de compras, seguimiento de órdenes, análisis de gastos |
| 💰 Ventas         | Control de inventario de productos, precios de venta, seguimiento de pedidos, análisis de ingresos                            |
| 🏬 Almacén        | Entradas/salidas de inventario, conteos, stock en tiempo real, kardex, stock de seguridad, análisis de antigüedad             |
| 📊 Inventario     | Control detallado, costos de compra, costos de venta, análisis de utilidad                                                    |
| 💵 Finanzas       | Caja, bancos, cuentas por cobrar/pagar, análisis de utilidades                                                                |
| 📁 Proyectos      | Gestión de proyectos, materiales, seguimiento de producción                                                                   |
| 🏗️ Activos fijos | Registro de activos, cálculo de depreciación                                                                                  |
| 👨‍💼 Nómina      | Gestión de empleados, reportes salariales                                                                                     |
| 🗂️ Oficina       | Gestión documental, información interna, procesos administrativos                                                             |
